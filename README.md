# دار الحياة - نظام إدارة مركز العلاج النفسي وعلاج الإدمان

## نبذة عن المشروع
نظام إدارة متكامل لمركز "دار الحياة" للعلاج النفسي وعلاج الإدمان، مصمم لتبسيط إدارة المرضى، والمالية، والموظفين، والمجموعات العلاجية مع دعم متقدم للغة العربية.

## المميزات الرئيسية
- إدارة كاملة لبيانات المرضى: السجلات الطبية، مدة الإقامة، الرسوم
- نظام مالي متكامل: تسجيل المصاريف، التحصيلات، التقارير المالية
- إدارة الموظفين: سجلات الموظفين، الرواتب، المكافآت
- المجموعات العلاجية: جلسات العلاج الجماعي، تقارير التقدم العلاجي
- نظام صلاحيات متعدد المستويات لمختلف الأدوار الوظيفية
- دعم كامل للغة العربية وواجهة مستخدم من اليمين إلى اليسار

## المتطلبات التقنية
- Python 3.11+
- Flask framework
- PostgreSQL
- المكتبات المطلوبة موجودة في ملف `requirements-render.txt`

## إعداد بيئة التطوير
1. انسخ المستودع:
```
git clone https://github.com/your-username/dar-el-hayah.git
cd dar-el-hayah
```

2. أنشئ بيئة افتراضية (اختياري ولكن موصى به):
```
python -m venv venv
source venv/bin/activate  # لينكس/ماك
# أو
venv\Scripts\activate  # ويندوز
```

3. قم بتثبيت المكتبات المطلوبة:
```
pip install -r requirements-render.txt
```

4. أنشئ ملف `.env` وأضف المتغيرات البيئية المطلوبة (راجع ملف `.env.example`):
```
DATABASE_URL=postgresql://username:password@host:port/database
SESSION_SECRET=your-secret-key
```

5. قم بتشغيل التطبيق:
```
python main.py
```

## النشر على Render
1. قم بدفع المشروع إلى GitHub
2. سجل الدخول إلى [Render](https://render.com/)
3. اختر "New Blueprint" وحدد مستودع GitHub
4. ستقوم Render تلقائيًا بإنشاء خدمة ويب وقاعدة بيانات باستخدام ملف `render.yaml`
5. أضف المتغيرات البيئية المطلوبة في إعدادات Render

## هيكل المشروع
- `app.py` - تكوين تطبيق Flask والإعدادات
- `main.py` - نقطة بدء التطبيق
- `models.py` - نماذج قاعدة البيانات
- `routes.py` - مسارات التطبيق
- `auth.py` - وظائف المصادقة والتفويض
- `forms.py` - نماذج WTForms
- `templates/` - قوالب HTML
- `static/` - ملفات CSS و JavaScript والصور
